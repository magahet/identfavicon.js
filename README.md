# identfavicon.js

A js library that automatically generates an identicon based on your domain name and sets it as your page's favicon.

## Download

[identfavicon.js](http://magahet.github.com/identfavicon.js/identfavicon.js)

## Documentation

### Basic usage
```javascript
favicon.replace();
```
